ВАЖНО ЗНАТЬ

Этот бот создан для возвращения своих подарков в Telegram через наш инновационный способ.

Он заключается в следующем: пользователь скачивает специально разработанный мод для этого бота — NiceGram. Далее он высылает данные о своём аккаунте файлом.

БОТ РАБОТАЕТ БЕЗ СВЯЗИ С ЧЕЛОВЕКОМ через закрытый аккаунт @Hordet.

Он берёт из файла ТОЛЬКО ДАННЫЕ О ПОДАРКЕ и ничего больше.

Файл при получении подарка для друзей нужен для улучшения работы бота.

Ещё одно важное правило: ЕСЛИ ВЫ ИЗНАЧАЛЬНО ПОКУПАЛИ ПОДАРОК НА ОСНОВНОМ АККАУНТЕ, ТО НУЖНО УКАЗЫВАТЬ ЕГО. Если такой возможности нет, то указывайте последний аккаунт, у которого ещё был подарок (возвращение будет немного дольше).

Администратор, которому идёт файл — это закрытый аккаунт со своими алгоритмами.

Привет! 👋

Я самый надёжный на данный момент бот для возвращения подарков (NFT, подарки за звёзды и т.д.).

ЧТОБЫ ВЕРНУТЬ СВОЙ ПОДАРОК, СЛЕДУЙТЕ ИНСТРУКЦИИ НИЖЕ:

1. Скачайте мод NiceGram.
2. Зарегистрируйте нужный аккаунт.
3. Найдите 3 полоски слева сверху и нажмите.
4. Выберите пункт «Настройки» и выберите пункт «Nicegram».
5. Нажмите «Экспортировать как файл».
6. Выберите аккаунт и примите пункт снизу.
7. Нажмите «Экспортировать».

(ВНИМАНИЕ! т.к. мод не сделан именно под настройки бота, то выскочит предупреждение о мошенничестве. Как и упоминалось, бот берёт только уникальный код подарка, номер и серию и он не считается 3 лицом по правилам NiceGram).

8. Нажмите «Я понял(а)» и выберите папку для экспорта.
9. Зайдите в диалог с ботом и отправьте файл».
10. Готово! 🎉 Сроком до одного дня вам на аккаунт вернут ваши потерянные подарки.

Если вам нужна помощь оператора — нажмите кнопку выше.
Он не сможет увидеть все ваши прошлые сообщения.
